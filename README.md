# USCIS-Tracker

This repository belongs to `piranfar` and contains a GitHub Pages starter site for the `USCIS-Tracker` project.

## Deploy instructions

1. Make sure the repository is named `USCIS-Tracker` under your GitHub account.
2. Copy this project to the repository.
3. Commit and push the files.
4. In GitHub repository `Settings` → `Pages`, select:
   - Source: branch `main`
   - Folder: `/ (root)`
5. Save and wait a few minutes.

Your site will be available at:

`https://piranfar.github.io/USCIS-Tracker/`
